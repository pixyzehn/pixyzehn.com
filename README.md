# [pixyzehn.com](https://pixyzehn.com)

## Requirements

- [Node.js](https://nodejs.org) >= 18.20.8

## Installation

```bash
npm install
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run dev
```
